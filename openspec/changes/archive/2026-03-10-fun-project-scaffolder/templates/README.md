# Embedded Template Contents

This directory contains the actual content for each embedded template. Use these as reference when implementing the C string templates in the fun binary.

## Templates

1. `src_main.c.tmpl` - Main entry point
2. `src_cli.h.tmpl` - CLI header
3. `src_cli.c.tmpl` - CLI implementation
4. `commands_cmd_version.h.tmpl` - Version command header
5. `commands_cmd_version.c.tmpl` - Version command implementation
6. `commands_cmd_help.h.tmpl` - Help command header
7. `commands_cmd_help.c.tmpl` - Help command implementation
8. `build_windows_amd64.bat.tmpl` - Windows build script
9. `build_linux_amd64.sh.tmpl` - Linux build script
10. `fun.ini.tmpl` - Project manifest
11. `README.md.tmpl` - Project documentation
12. `arch_startup_windows_amd64_windows.c.tmpl` - Windows startup
13. `arch_startup_linux_amd64_linux.c.tmpl` - Linux startup
